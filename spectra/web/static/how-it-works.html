How Spectra Works

Spectra reads a forward-facing driving video frame by frame, builds a picture of the road and the traffic in it, and asks one question for every object: is this on a collision course with my lane? The map below is the full pipeline. Open any stage or the topic menu for the details.

Motion + Depth

Solid Internal motion/depth step Brighter Used downstream
Optical flow
flow on a downscaled pair (long side ≤ 320 px), upscaled back
ego-motion: RANSAC homography on LK corners, then subtracted
flow = DIS(current_gray, previous_gray) − ego_motion
magnitude_norm = normalize(|flow|)
divergence_norm = normalize(divergence(flow))
motion_signal = 0.65 × magnitude_norm + 0.35 × divergence_norm
Depth
depth_m = clamp(depth_model(rgb), 0, 80 m)
near_map = 1 − depth_m / 80
crop = bbox lower-center region (x 15%..85%, y 30%..100%)
distance_m = percentile(valid crop, 25)
refresh if first frame, frame % depth_every = 0, or quick_motion_risk > 0.15
Longitudinal Kalman
state = (distance, range_rate); closing_mps = −range_rate
measurement_var = (0.05 × distance_m)² + 0.5²
reject sample if innovation² > 9 σ² (3σ gate) → coast
ttc_depth = distance_m / closing_mps
valid when updates ≥ 2 and closing_mps ≥ 0.30
depth_confidence = speed_term × freshness × velocity_certainty

Lane Pipeline

Solid Internal lane step Brighter Used downstream
Lane position
lane_center_x = midpoint(left_lane, right_lane) at bbox bottom
lane_half_width = lane_width_at_bbox_bottom / 2
lane_pos = (bbox_center_x − lane_center_x) / lane_half_width
0 = center, ±1 = boundary, ±2 = one lane away
Crossing risk
lateral_v = median Δlane_pos / Δt over recent track history
predicted_pos = lane_pos + lateral_v × min(fused_ttc, 3)
margin = max(0, |predicted_pos| − 0.6)
lane_trust = clamp((lane_conf − 0.25) / 0.60, 0, 1)
far_reliability = 0.40 + 0.60 × depth_frac
predicted = exp(−margin² / 0.25) × lane_trust × far_reliability
crossing_risk = clamp(max(corridor_overlap, predicted), 0, 1)

Per-Object Risk Evaluation

Solid Internal risk step Brighter Used downstream
Time-to-collision
ttc_depth = distance_m / closing_mps (physical ETA)
ttc_expansion = 1 / expansion_rate when expansion_rate ≥ 0.01
ttc_flow = distance_from_vp / radial_p75 × flow_dt
valid TTC cues are clamped to [0.15s, 15.0s]
fused_ttc = weighted_median(valid cues, cue_confidence)
eta_pressure(ttc_depth) = clamp((3 − ttc_depth) / 3, 0, 1)
Cue agreement
values = valid(ttc_depth, ttc_flow, ttc_expansion)
spread = (max(values) − min(values)) / median(values)
ttc_agreement = clamp(1 − spread, 0, 1)
if fewer than 2 valid cues: ttc_agreement = 1
Proximity + approach
proximity = 1 if distance_m ≤ 8
proximity = 0 if distance_m ≥ 60
proximity = clamp((60 − distance_m) / 52, 0, 1) otherwise
metric = clamp((closing_mps − 0.30) / (12.0 − 0.30), 0, 1)
expansion = clamp(expansion_rate / 0.6, 0, 1)
flow = clamp(flow_magnitude_p75, 0, 1)
approach = class_weight × (0.50 × metric + 0.30 × expansion + 0.20 × flow)
Confidence (gate input)
raw_confidence = clamp(0.70 × detection_confidence + 0.30 × lane_confidence, 0, 1)
agreement_gate = 0.70 + 0.30 × ttc_agreement
confidence = EMA(raw_confidence × agreement_gate)
detection_confidence still controls the low-confidence SAFE override
Risk score
signal = 0.40 × eta_pressure + 0.30 × proximity + 0.25 × approach + 0.05 × brake
gate = 0.65 + 0.35 × confidence
relevance = crossing_risk
risk.score = round(gate × relevance × signal, 3)
State banding
raw_state = DANGER if risk.score ≥ 0.60
raw_state = CAUTION if risk.score ≥ 0.25
raw_state = SAFE otherwise
escalate → DANGER if ttc_depth < 1.0s in ≥2 recent frames and risk.score ≥ 0.25, and the imminent reading is corroborated (distance ≤ 12m, or the three TTC cues agree ≥ 0.5) — blocks a young depth-Kalman track's non-physical closing speed from snapping a distant vehicle to DANGER
force → SAFE if detection_confidence < 0.20
raw_state is stabilized later in Decision + Delivery

Decision + Delivery

Solid Internal decision step Brighter Used downstream
Primary object
eligible if state ≥ CAUTION, or |lane_pos| ≤ 1
SAFE off-corridor eligible only when moving inward and predicted inside lane
primary = argmax eligible objects by
(risk.score, state_severity, eta_pressure)
State stabilization
banner is stabilized from the raw per-object states; the shown primary object/score always stay on the real current object (never a frozen or dropped-out value)
dropout hold: if an approaching near in-corridor threat was the primary within the last 1.0s but is momentarily absent, hold only the banner BAND at CAUTION so a brief miss cannot flash SAFE — released as soon as that threat is seen receding
near in-corridor test: close + within one lane of the ego path, with a geometry escape — a wide, bottom-anchored box counts as a lead in our path even when its lane_position snaps to the ±1.5 clamp, so a tailgated lead reading far-left/right is not mistaken for side traffic
upgrade after 3 consecutive frames
downgrade after 7 consecutive frames, or 3 when the scene has cleared (no approaching in-corridor threat) so a passed threat's DANGER does not linger
confirmed ttc_depth ≤ 1.0s DANGER bypasses upgrade delay
Saved events
event_score = primary_risk_score if present, else risk_score
dedup if |timestamp − saved.timestamp| ≤ 1.0s
stronger event_score replaces weaker event in the window
keep top-N saved events by event_score
render overlays only after dedup + top-N retention
