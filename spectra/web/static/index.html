How Spectra Works

          

        

        
      


      

        

          

            
            

              

                
                
01

              

              
Data Ingestion

              
Frames are streamed through a WebSocket for real-time preview while the backend initializes the vision
                stack.

              

                FastAPI
                WebSockets
              

            


            
            

              

                
                
02

              

              
Road Geometry

              
Estimates lane corridor and vanishing point using UFLDv2. Normalizes positions relative to the
                ego-lane.

              

                Lane ROI
                Vanishing Point
              

            


            
            

              

                
                
03

              

              
Depth (Spatial)

              
Depth Anything V2 produces relative nearness maps. CoreML acceleration is used on Apple Silicon for
                high FPS.

              

                ONNX
                CoreML
              

            


            
            

              

                
                
04

              

              
Motion Analysis

              
DIS Optical Flow calculates pixel-level vectors. RANSAC-based homography compensates for camera
                ego-motion.

              

                OpenCV DIS
                Ego-Motion
              

            


            
            

              

                
                
05

              

              
Object Tracking

              
YOLO detects road participants. A lightweight IoU tracker maintains history for velocity and expansion
                rates.

              

                YOLOv8
                IoU Tracker
              

            


            
            

              

                
                
06

              

              
Risk Fusion

              
Fuses TTC from three sources: BBox Expansion, Radial Flow, and Depth-Delta using weighted median logic.
              

              

                TTC Fusion
                Median Filter
              

            


            
            

              

                
                
07

              

              
Decision Engine

              
Classifies state as SAFE, CAUTION, or DANGER. Hysteresis stabilization prevents jitter in risk
                transitions.

              

                Hysteresis
                State Machine
              

            


            
            

              

                
                
08

              

              
Efficiency

              
Deferred rendering ensures heavy overlays are only drawn for saved events or on-demand previews.

              

                Telemetry
                Deferred Render
